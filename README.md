# O Campo que Alimenta o Mundo e Preserva o Futuro🌱
Projeto desevolvido para o concurso Agrinho 2026, com o tema: "Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente"; com maior perspectiva do estado paranaense

---

## 📌 Objetivo do Trabalho
O objetivo é criar um site acessível e responsivo que mostre como o Paraná e o Brasil já unem tecnologia e sustentabilidade no campo. O projeto usa a programação para valorizar o agronegócio moderno e garantir a inclusão digital de todos os usuários.

---

## 🛠️ Recursos Técnicos Utilizados
### 💻 Tecnologias
* **HTML5 (Estrutura Semântica)**: Organização do conteúdo utilizando tags semânticas (*header, main, section, figure*), que facilitam a leitura do código e melhoram o ranqueamento SEO.
* **CSS3 (Estilização e Responsividade)**: * Criação de uma identidade visual baseada em variáveis CSS (:root), utilizando tons de marrom e verde que remetem ao solo e à natureza.
> Uso de Media Queries ou layouts flexíveis para garantir que o site se adapte a computadores, tablets e celulares (responsividade).
* **JavaScript Vanilla (Interatividade)**: Manipulação do DOM para criar a lógica de alternância de temas (Modo Claro e Modo Escuro) em tempo real.
### ♿ Recursos de Acessibilidade Implementados
1. **Atributos ARIA e Semântica**: Uso de aria-label no botão de alternância de tema para orientar leitores de tela.
2. **Textos Alternativos**: Inclusão da propriedade alt detalhada em todas as imagens para usuários com deficiência visual.
3. **Alto Contraste**: Paleta de cores planejada para manter uma excelente legibilidade entre o texto e o fundo, tanto no modo claro quanto no escuro.
4. **Navegação por Teclado**: Estilização do foco (outline) para garantir que usuários que não usam o mouse consigam saber exatamente onde estão navegando na tela.
